Light Film Maker is a program for creating light animations

Why GPLv3?
We believe that animation tools should remain open and accessible. By using GPLv3, we ensure that:
The source code of this LFM stays open forever.
Any derivative works or "propprietary clones" must also release their source code.
Users are protected from patent-related claims.
